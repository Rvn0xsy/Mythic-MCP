# Getting Started

Welcome to the Mythic MCP Server setup guide. Choose the path that fits
your environment.

## Prerequisites

- **Go 1.23+** (to build from source)
- **Mythic C2 ≥ 3.3.0** running and accessible
- A Mythic **API token** or admin credentials
